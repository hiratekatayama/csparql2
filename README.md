# csparql2
C-SPARQL Engine 2.0

Implemented using Yasper Common Library


```xml
 <repositories>
   <repository>
    <id>jitpack.io</id>
       <url>https://jitpack.io</url>
   </repository>
 </repositories>
 <dependencies>
    <dependency>        
	<groupId>com.github.riccardotommasini</groupId>
	<artifactId>csparql2</artifactId>
	<version>2.3.0</version>
    </dependency>
</dependencies>
```
